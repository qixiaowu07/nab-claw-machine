# 拿捏 NAB - AI 看透娃娃机 · 使用说明

> 用 AI 帮你分析娃娃机，轻松拿捏每一爪！

---

## 🚀 快速开始

请解压ZIP包后：

1. 用浏览器直接打开 `index.html` 即可使用（无需安装、无需服务器）
2. 默认内置**智能演示模式**，开箱即用，所有功能可直接体验
3. 如需接入真实 AI 大模型分析，可配置智谱 AI API Key（GLM-4-Flash 永久免费）

---

## 🔑 配置真实 AI（可选）

本应用支持接入**智谱 AI GLM-4-Flash**（永久免费大模型）：

1. 访问 [https://open.bigmodel.cn/](https://open.bigmodel.cn/) 注册账号
2. 进入「API Keys」页面，创建新的 API Key
3. 打开拿捏 App → 底部「👤 我的」→「⚙️ AI设置」
4. 粘贴 API Key，点击「保存」即可

> 💡 **不配 Key 也能用！** 默认智能演示模式内置 14 种热门娃娃数据库，支持关键词智能匹配，开箱即用。

## ✨ 功能全览

### 🏠 首页
- **立即拍照抓诊**：一键进入拍照识别模式
- **描述抓诊入口**：不方便拍照？文字描述也能生成策略
- **今日数据**：今日帮助玩家数、平均胜率提升等统计
- **功能快捷入口**：附近好抓的店、社区热门战报、AI收益预估

### 🎯 AI 策略（核心功能）

#### 📸 拍照抓诊
- 点击首页「📸 立即拍照抓诊」进入
- 默认已预置演示照片，直接点击快门即可体验 AI 分析
- 可点击「📁 从相册选择」上传自己拍摄的娃娃机照片
- AI 分析过程：
  - 扫描线动画 + 识别框高亮，实时显示识别进度
  - 自动识别娃娃分布、推荐最佳抓取点
  - 标记 AI 推荐抓取位置（带动态圆圈标记）
- 分析结果包括：
  - ✅ 推荐目标娃娃及具体抓取位置
  - ✅ 成功率预估（百分比）+ 难度评级
  - ✅ 建议下爪角度、预估抓取次数
  - ✅ 机器状态判断（挡板高度、爪力强弱）
  - ✅ AI 操作建议（详细步骤）
  - ✅ 可标记正版娃娃，预估二级市场收益

#### 💬 描述抓诊
- 首页点击「描述抓诊」卡片进入 AI 对话页面
- 用自然语言描述娃娃机情况（娃娃形状、挡板高低、爪力强弱、有无强力爪）
- AI 头像为品牌吉祥物"拿捏"，对话式交互
- 提供快速描述按钮，一键选择常见场景
- AI 同样给出完整抓取策略分析
- 支持多轮对话，可追问细节

### 💬 社区
- **战报分享**：查看其他玩家的抓娃战果
- **机器参数**：各门店机器参数评级
- **避坑指南**：真实玩家踩坑经验
- 支持👍👎投票、点赞有用内容
- 可发布自己的战报到社区

### 🗺️ 地图
- 查找附近娃娃机门店
- 门店标记：好抓推荐（绿色）/ 强力抓（橙色）/ 避雷（红色）
- 一键导航到店
- 附近好店推荐列表

### 👤 我的
- **战绩统计**：总抓取次数、成功率、节省币数、二级市场收益
- **AI 设置**：配置/重置智谱 AI API Key
- **收藏门店**：收藏的好抓门店

## ⚠️ 注意事项

1. **API Key 安全**：Key 保存在本地浏览器 localStorage 中，不会上传到任何服务器
2. **免费额度**：智谱 GLM-4-Flash 永久免费，足够日常使用
3. **网络要求**：配置 API Key 后需联网调用 AI；演示模式完全离线可用
4. **浏览器兼容性**：推荐使用 Chrome / Safari / Edge 最新版本
5. **拍照功能**：从相册选择照片支持所有浏览器；摄像头功能需要 HTTPS 或 localhost 环境
