# 🎯 NAB 拿捏 — AI 看透娃娃机

> 用 AI 帮你分析娃娃机，轻松拿捏每一爪！

**拿捏 NAB** 是一款 AI 驱动的抓娃娃策略助手。通过拍照或文字描述娃娃机现场情况，AI 自动识别娃娃分布、分析挡板高度与爪力强弱，为你推荐最佳抓取目标、操作手法，并可标记正版娃娃预估二级市场收益。

- 🧠 **AI 双模式**：拍照抓诊（视觉识别）+ 描述抓诊（对话式分析）
- 📸 **开箱即用**：已内置智谱AI GLM-4-Flash共享Key，默认使用真实AI分析
- 🆓 **免费 AI**：智谱 AI GLM-4-Flash（永久免费大模型），也可配置自己的Key
- 🎮 **完整产品体验**：5 大 Tab、开场动画、新手引导、品牌 IP
- 🌐 **纯前端**：单个 HTML 文件即可运行，零后端依赖

## ✨ 功能特性

### 🎯 AI 智能抓诊（核心）

- **📸 拍照抓诊**：拍摄/上传娃娃机照片，AI 自动扫描识别娃娃分布与位置，实时展示扫描动画、识别框和目标高亮，最终生成抓取策略
- **💬 描述抓诊**：用自然语言描述娃娃机情况（娃娃形状、挡板高低、爪力强弱等），AI 对话式给出策略建议，支持多轮追问
- **策略输出**：推荐目标 + 抓取位置 + 成功率 + 难度评级 + 操作步骤 + 预估成本 + 注意事项
- **正版识别**：可标记正版 IP 娃娃（三丽鸥、迪士尼、宝可梦等），预估二级市场价格与收益

### 🏠 完整 App 体验

- 🏠 **首页**：拍照/描述抓诊入口、今日数据统计、快捷功能
- 💬 **社区**：战报分享、机器参数评级、避坑指南、👍👎 投票
- 🗺️ **地图**：附近好店推荐、门店标记（好抓/强力抓/避雷）、一键导航
- 👤 **我的**：战绩统计、AI 设置、收藏门店、分享 App
- 🎬 **开场动画**：品牌吉祥物 + Splash 启动动画
- 🧭 **新手引导**：5 步交互引导，快速上手

## 🔑 配置自己的 API Key（可选）

应用**默认已内置共享Key**，开箱即用真实AI分析，API调用失败时自动回退智能演示模式（内置14种热门娃娃数据库+关键词智能匹配）。

如共享Key额度用尽，您也可以配置自己的智谱AI Key（GLM-4-Flash **永久免费**）：

1. 访问 [智谱 AI Open Platform](https://open.bigmodel.cn/) 注册账号
2. 进入「API Keys」页面创建新的 API Key
3. 打开 App → 底部「👤 我的」→「⚙️ AI 设置」
4. 粘贴 API Key 并保存，或点击「恢复默认」切回内置共享Key

API Key 仅保存在浏览器本地 localStorage 中，不会上传到任何服务器。
